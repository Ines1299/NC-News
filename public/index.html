NC NEWS API

An API of NC NEWS with everything northcoders!

Available endpoints:

  • GET /api/topics

    Retrieve a list of all topics.

    Example response:

      
{
  "topics": [
    {
      "slug": "coding",
      "description": "Code is love, code is life"
    },
    {
      "slug": "football",
      "description": "FOOTIE!"
    },
    {
      "slug": "cooking",
      "description": "Hey good looking, what you got cooking?"
    }
  ]
}

  • GET /api/articles/:article_id

  • POST /api/articles/:article_id/comments

    Post a comment under a specific article_id.

    Example of how to write in the body to be able to post a comment:

      
		{
			"username": "grumpy19",
			"body": "I am a comment",
			
		}
  
   
    Example of retrieved comment:
    
          
      
		{
			"comment_id": 203,
			"votes": 0,
			"created_at": "2026-02-10T17:21:00.000Z",
			"author": "grumpy19",
			"body": "I am a comment",
			"article_id": 2
			
		}
